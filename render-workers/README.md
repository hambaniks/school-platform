# Render Workers

Cron workers for SchoolNet v3.0 are located in the `render-cron/` directory:

- `render-cron/workers/attendance-compliance.mjs`
- `render-cron/workers/health-pdf-generator.mjs`
- `render-cron/workers/truancy-watchdog.mjs`

See `render.yaml` at the project root for deployment configuration.
